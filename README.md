## Что делает эта программа
- Поможет привести к единому формату большое количество изображений в разных форматах. 
- Преобразует изображения форматов png, bmp, gif, avif, webp в формат jpg.
- После конвертации переименовывает файлы в последовательной нумерации (например, 1.jpg, 2.jpg и так далее).
- Перемещает все переименованные файлы в указанную папку.

---

## Как использовать

1. Укажите путь к следующим папкам: 
   - `source_folder` — папка исходных изображений для конвертации
   - `result_folder` — папка с результатами после процесса конвертации

2. Создайте и активируйте окружение

3. Запустите скрипт

---

## Требования
- Python 3.x

Установка библиотек:
- pip install pillow
- pip install pillow-avif-plugin
